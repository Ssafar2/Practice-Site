Açıqlama

Arkadaşlar herkese merhabalar, Bu seri'de sizlerle beraber günümüzde de popülerliğini korumayan devam eden , Türkiye 'de ve dünyada halen güncelliğini koruyan ve birçok alanda halen kullanılan Java programlama dilini öğreneceğiz. İş ilanlarında java developer sıklıkla aranmaktadır. İş ilandaki şartları karşılayabilmek için öncelikle Java SE(Standart Edition) dediğiniz java'nın core yapısını anlamak gerekiyor. Java SE öğrenilmeden hiçbir framework , kütüphane öğrenilemez. İşte bizde bu eğitim serimizde Javanın core yapısını öğrenmeye çalışacağız. Arkadaşlar bu eğitimle beraber java core yapısını öğrendikten sonra artık java ile proje geliştirirken ihtiyaç duyduğunuz frameworkleri ve kütüphaneleri de kullanabilir. Daha complex projeler çıkarabilirisiz. Bu eğitimi bitirme süreniz 4 ile 6 ay arasında değişmeli . Çok yavaş ilerlerseniz öğrendikleriniz unutmuş olacaksız . Çok hızlı ilerlerseniz bu seferde konular oturmayacaktır. Belirttiğim süreye uymanızı tavsiye ederim.

Peki Neler Öğreneceğiz

1-Programlama dillerinin çalışma mantığını anlamış olacaksınız.

2-Java standart edition bilgisi öğreneceksiniz.

3-Nesne yönelimli programlamanın ne demek ve nasıl kullanıldığını öğreneceksiniz

4-Algoritma bilginizi geliştireceksiniz.

5-Javanın web ve mobil programlama frameworklerine alt yapısı kazanmış olacaksınız
